# python-web-fastapi-api-dolt-tortoise-orm-simple

## Description
Simple web app that serves api
for a fastapi project.

Uses tortoise query a table `dog`.

## Tech stack
- python
  - fastapi
  - uvicorn
  - tortoisee
- bootstrap
- jquery
- dataTable
- dolt

## Docker stack
- python:latest
- dolthub/dolt-sql-serverdb:latest

## To run
`sudo ./install.sh -u`
- [Availble at](http://localhost/dog)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
